# enigmacc

Hey there,  I have "recently" been blessed by the beauty of [enigmatics](https://enigmatics.org/) puzzles,  and also by the amazing [IOCCC](https://en.wikipedia.org/wiki/International_Obfuscated_C_Code_Contest) artwork I found,  so that's why I took the worst decision of my life,  to make an IOCCC-style C program inspired on neocities.

<br>

`enigmacc` picks a random playable puzzle from enigmatics for a user to solve.  It will then bless you with a pink !@)@#&%@$@ and a "#(?%#&#@ message.

<br>

```sh
gcc -O2 enigmacc.c -o enigmacc
./enigmacc <user>
```
